# CareerOS — Scoring Rules

## Score Range: 0–100

| Category             | Max Points | Criteria |
|----------------------|-----------|----------|
| Title Alignment      | 20        | Does the job title match target role types? |
| Salary Alignment     | 20        | Is salary in the $80K–$130K target range? |
| Remote Eligibility   | 15        | Is the role fully remote (15), hybrid (8), or on-site (0)? |
| Experience Match     | 20        | Is experience level 1–5 years, mid-level, or appropriate? |
| Skill Match          | 15        | How many candidate skills appear in the job description? |
| Growth/Path Value    | 10        | Does the role offer learning, leadership, or career path value? |

---

## Score Labels

| Label       | Range    | Meaning |
|-------------|----------|---------|
| 🟢 Green Light  | 80–100   | Strong match. Generate packet immediately. Prepare to apply. |
| 🟡 Yellow Light | 65–79    | Worth reviewing. May need tailoring or more context. |
| 🔴 Red Light    | Below 65 | Weak match or hard filter triggered. Auto-reject or manual review. |

---

## Hard Filters (Auto-Reject)

Jobs are automatically rejected before scoring if they match any of these:
- Commission-only compensation
- Relocation required (on-site only)
- 8–10+ years required in a single narrow technical title (e.g., "Senior ML Engineer 10+ years")
- PhD or Doctorate required
- Advanced ML/AI engineering tools (PyTorch, CUDA, LLM training)
- Heavy software engineering (Senior Software Engineer, Full-Stack Engineer)
- No remote option available
- Salary max below $72,000 (when salary is visible)

---

## Resume Draft Rules

- Draft resume ONLY for Green Light (80+) jobs automatically
- For Yellow Light, draft only if manually requested
- Never draft for Red Light jobs
- All claims must be truthful — never invent experience, tools, certifications, or metrics

---

## Scoring Notes

- If salary is not listed, the system assigns a neutral score (10/20) for salary
- If posted date is unclear, the system does NOT invent a date — marks as "uncertain"
- If remote status is unclear, the system marks as "unknown" and scores conservatively
- Experience level "unspecified" gets a moderate score (13/20) by default
