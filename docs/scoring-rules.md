# CareerOS Scoring Rules

## Score Range: 0-100

| Category | Max Points | Narrowed Criteria |
| --- | ---: | --- |
| Title Alignment | 20 | Strongest fit: operations, process improvement, operational excellence, implementation, project/program coordination, customer operations, onboarding/enablement ops, workflow automation, reporting/KPI, digital/eCommerce operations. Secondary titles score lower unless the description clearly includes operations/process/project/reporting context. |
| Salary Alignment | 20 | Best fit is visible salary in the $80K-$130K range. Missing salary is conservative, not neutral. |
| Remote Eligibility | 15 | Fully remote only for normal MVP scans. Hybrid/on-site stays low or is filtered unless manually reviewed. |
| Experience Match | 20 | Best fit is associate to mid-level, roughly 1-5 years. Senior, director, principal, staff, VP, or 7+ year roles are poor fit for this transition search. |
| Skill Match | 15 | Rewards confirmed strengths: operations, SOPs, training, compliance, budget/vendor coordination, Excel, Power BI portfolio, reporting, KPIs, workflow digitization, implementation, onboarding, customer feedback, documentation, and cross-functional work. |
| Growth/Path Value | 10 | Rewards roles that build toward remote operations, systems improvement, implementation, reporting, automation, and cross-functional project ownership. |

## Score Labels

| Label | Range | Meaning |
| --- | --- | --- |
| Green Light | 80-100 | Strong fit. Generate packet immediately. Prepare for manual review/apply. |
| Yellow Light | 65-79 | Worth reviewing. May need extra context or careful tailoring. |
| Red Light | Below 65 | Weak fit or hard filter triggered. Reject or review only if there is a special reason. |

## Strongest Fit Titles

- Business Operations Analyst / Manager
- Operations Analyst / Manager
- Strategy & Operations Analyst
- Business Process Analyst
- Process Improvement / Continuous Improvement / Operational Excellence
- Operations Project Manager / Project Coordinator
- Implementation Project Manager / Specialist / Coordinator
- Program Manager / Program Coordinator / PMO Analyst
- Customer Operations / Customer Success Operations / Client Operations
- Onboarding / Enablement / Training Operations
- Business Systems / Workflow Automation / Process Automation / No-Code Automation
- Reporting Analyst / KPI Analyst / Performance Analyst / Operations Data Analyst
- Digital Operations / eCommerce Operations
- SOP Documentation / Quality Operations

## Secondary Fits

These can be good only when the description clearly includes operations, process, reporting, implementation, onboarding, workflow, SOP, Excel/Power BI, vendor, compliance, or cross-functional project work:

- Business Analyst
- BI Analyst / Business Intelligence Analyst
- Data Analyst / Insights Analyst
- Customer Success Manager / Client Success Manager
- Revenue Operations / Sales Operations / CRM Operations
- Product Operations / Marketplace Operations / Content Operations
- Supply Chain / Procurement / Inventory / Vendor Management
- Compliance Analyst / Quality Assurance Analyst
- Learning & Development / Workforce Operations

## Avoid or Auto-Reject

- Account Executive, SDR, BDR, Business Development, sales quota roles
- Generic Account Manager roles
- Customer support/help desk roles unless explicitly operations-focused
- Recruiting, legal, accounting, tax, audit, finance-only roles
- Software, platform, DevOps, security, ML, data engineering, analytics engineering, or solutions engineering roles
- Product Manager roles unless they are clearly Product Operations
- Director, VP, Head of, Principal, Staff, or 7+ year senior roles
- Senior/Sr. titles by default, even when the function is operations or program management
- Technical Project Manager / Technical Program Manager roles unless manually approved
- Non-US remote/location-specific roles such as Canada-only, Chile, UAE, EMEA, APAC, UK, or Europe-only
- Commission-only compensation
- Relocation required or on-site only
- PhD/doctorate required
- Heavy advanced coding, ML, or engineering requirements
- Salary max below $80K when visible

## Resume Draft Rules

- Draft resume only for Green Light jobs automatically.
- Draft Yellow Light jobs only when manually requested.
- Never draft for Red Light jobs.
- Keep every claim truthful. Do not invent experience, tools, certifications, employers, degrees, or metrics.
