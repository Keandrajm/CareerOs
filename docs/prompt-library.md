# Prompt Library Template

Do not commit personal application answers here.

Keep reusable answer patterns generic:

- Why are you interested in this role?
- Why are you a good fit?
- Tell us about yourself.
- Salary expectations.
- Work authorization.
- Remote work experience.
- Project management experience.
- Data/reporting experience.
- Customer success or operations experience.

CareerOS generates specific drafts from the private candidate profile at runtime.
